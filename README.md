# Image Watermark Tool

A desktop application that lets you add custom text watermarks to your pictures. It provides an easy-to-use interface to select an image, type out your text, and save a copy to your computer.

## Features

* **Smart Font Scaling:** Automatically sizes the text watermark so it fits perfectly regardless of how large or small your image is.
* **Background Contrast Options:** Switch between light and dark text modes to ensure your watermark stays completely visible on any background.
* **Transparent Blending:** Applies text at a clear, semi-transparent opacity to avoid blocking the image.

## How to Run

1. Clone this repository or download the source code.
2. Ensure you have Python 3 and the Pillow library installed.
3. Run the application from your terminal or IDE:

```bash
python main.py
